# Ideal Body

## Tentang Platform
  Kesehatan tubuh yang ideal merupakan salah satu indikator penting dalam mewujudkan kesejahteraan hidup sesuai dengan tujuan SDG 3 â€“ Good Health and Well-being. platform ini merupakan web yang dapat membantu pengguna menghitung berat badan ideal, body mass index (BMI), basal metabolic rate (BMR), serta memberikan rekomendasi sederhana untuk mencapai berat badan yang sehat.

## Fitur Ideal Body
- Mengitung berat badan ideal user
- Menghitung body mass index dan basal metabolic rate
- Memberikan rekomendasi untuk berat badan ideal
- Calorie tracker untuk memperkirakan berat badan  

## How to use
1. Masukkan gender, umur, berat serta tinggi badan
2. kalkulasi nilai BMI dengan menekan tombol "Hitung BMI"
  Apabila ingin menggunakan fitur "Calorie Tracker", pengguna dapat melakukan Login terlebih dahulu sehingga fitur dapat diakses.
